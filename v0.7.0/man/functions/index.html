fluxes(prob)

Returns the lateral eddy fluxes within each fluid layer lateralfluxes1, ..., lateralfluxesnlayers and also the vertical eddy fluxes for each fluid interface verticalfluxes{3/2}, ..., verticalfluxes{nlayers-1/2}. (When nlayers=1 only the lateral fluxes are returned.)
